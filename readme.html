Requirements

For this project, please use React as your front-end framework.

On startup, the app should display a single button in the center of the screen. Pressing this button should cause the app to display a modal dialog styled to match the above mockup, driven by JSON data retrieved from an API endpoint (more on this below).

Clicking the 'X' in the upper right should dismiss the modal.

Clicking the 'RSVP' button should open the URL specified in the offers section of the JSON.

Clicking 'Get Directions' should launch Google Maps with directions to the event location. See Google's documentation for how to construct this link.

For the small map displayed within the event body, you may use any library. Leaftlet is a popular choice and supports OpenStreetMap, but you are welcome to use what you are most comfortable with.

Your app should retrieve the JSON for the event via an HTTP request. For convenience, we've uploaded event.json to mocky.io: https://www.mocky.io/v2/5d3752f1310000fc74b0788d.

Evaluation

We're looking for a faithful rendition of the attached mockup. While this project does not require complex logic, we're looking for idiomatic React code and a clean separation of concerns. We encourage you to use hooks! Unit tests are not required for this assignment.

To constrain scope, we haven't included requirements for details like hover and active states. You may implement whatever seems reasonable to you.

Similarly, it's fine if this UI is desktop-only. Bonus points if you make the UI responsive at smaller mobile screen sizes.

Feel free to use additional 3rd-party libraries as you see fit.

Deliverables

Email your completed solution, including unminified source code, as a tar.gz or zip archive to hr@pedago.com.

Please include any special instructions for how to run your project, especially if it uses a build system!
